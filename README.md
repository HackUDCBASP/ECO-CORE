# HACKUDC

## 📌 Descripción
ECO CORE es un videojuego open source educativo en 3D desarrollado íntegramente con Software Libre para la HackUDC 2026.
En esta aventura, encarnas a un agente de la Agencia de Recuperación Planetaria (ARP) en una misión crítica: guiar a los habitantes de un pueblo alienígena hacia la sostenibilidad. Utilizando Godot 4.3, el proyecto combina mecánicas interactivas con un mensaje ecológico profundo, fomentando la conciencia ambiental a través del asesoramiento estratégico a sus singulares personajes.

## 🚀 Características
- 🎮 Jugabilidad intuitiva y entretenida.
- 📚 Lecciones interactivas mediante diálogos y elecciones con consecuencias.
- 🎨 Estilo visual atractivo y relajante con gráficos 3D y muchas referencias.
- 🪐 Open Source.
- 💡 Desafíos variados que fomentan y refuerzan el aprendizaje.

## 🛠️ Tecnologías utilizadas
- **Motor de juego**: Godot 4.3
- **Lenguaje de scripting**: GDScript
- **Herramientas usadas para el modelado 3d, animaciones e ilustración**: Blender y Pixelorama
- **Plataformas objetivo**: Linux y Windows
- **Commits**: Se han usado Conventional Commits desde el inicio del proyecto hasta su versión actual

## 📥 Instalación y ejecución
## OPCIÓN 1
1. **Clonar el repositorio**
   ```sh
   git clone https://github.com/HackUDCBASP/ECO-CORE
   ```
2. **Abrir con Godot 4.3**
3. **Ejecutar el juego** desde el editor o exportarlo para tu plataforma preferida.

## OPCIÓN 2
1. **Descargar el ejecutable situado en /ejecutables**
2. **Disfrutar**
